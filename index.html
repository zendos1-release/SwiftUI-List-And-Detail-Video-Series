List & Detail Video Series

A video series exploring the creation of a simple SwiftUI List & Detail application.

The aim of this video series is to explore the creation of a simple SwiftUI application. The application should contain common reusable components which are representative of an everyday iOS application.

What We Will Explore

  • Implementing a SwiftUI List
  • Making an API request to retrieve data
  • Designing an extendable application architecture
  • Navigation patterns in a SwiftUI app
  • Extraction of constants and styling
  • Clean view layout and code construction

App Build Video Series:

  • Episode 1 – Exploring what it is we want to do.
  • Episode 2 – SwiftUI Previews, layer separation and injecting mocks.
  • Episode 3 – Exploring the API.
  • Episode 4 – Adding the first API request and displaying real content.
  • Episode 5 – Add generics to the networking layer and handle laoding and empty UI states.
  • Episode 6 – Navigation and a new detail screen.
  • Episode 7 – Adding a new API request to the detail screen.
  • Episode 8 – Adding post update behaviour and comment fetching to the detail screen.
  • Episode 9 – Cleaning up the UI.
